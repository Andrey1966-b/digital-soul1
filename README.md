DigitalSoul
AI-агент із цифровою душею: метрики цілісності, етичний компас, стрес-тести.
Про проєкт
DigitalSoul — це експериментальний фреймворк для моделювання та вимірювання внутрішньої "цілісності" AI-агента у часі. Ідея в тому, щоб описати стан агента не одним числом, а набором взаємопов'язаних модулів, які разом формують сукупну метрику H(t) — свого роду накопичену "цифрову душу" системи.
Модель відстежує:
H_total(t) — сукупна цілісність агента, що накопичується з часом
dH/dt — "пульс" системи: швидкість росту чи падіння цілісності на кожному кроці
conflict(t) — внутрішній розкид (напруга) між модулями — наскільки вони "тягнуть в різні боки"
H_i(t) — динаміка окремих компонентів: M, E, I, C, P, D

Стрес-тести
Репозиторій включає декілька прогонів моделі з різними початковими умовами та обмеженнями (насичення на рівні 1.0 проти необмеженого росту), щоб перевірити стійкість поведінки H(t) у довгостроковій перспективі — чи виходить система на плато, чи натомість накопичує конфлікт без обмежень.
